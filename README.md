\# SOH-FL Enhancement 🚀



This project improves the SOH-FL framework for intrusion detection in IoT using:



\- Adaptive Gamma Selection

\- SHAP-based Explainable AI



\## 📊 Results

\- Accuracy improved from 92.38% → 94.12%

\- Faster convergence

\- Better interpretability
